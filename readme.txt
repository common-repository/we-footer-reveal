== Description ==
The FAMOUS Footer reveal effect, now for WordPress ! üòç
Based on footer-reveal.js by Iain Andrew https://github.com/IainAndrew/footer-reveal

- Easy to use
In most cases, just install & activate !
- Customize
If the effect doesn't work just after activating the plugin, go to the customizer, then, you can change the "Footer Reveal selector" in "Footer Reveal Options".
(You see the result in live).

Elementor users :
This plugin is now compatible with Elementor.
To allow you edit your footer, reveal effect will be disabled in Elementor editor mode.
An indicator is displayed on the elements with reveal effect.

== Frequently Asked Questions ==

= I always see the footer with my content above =
This is because your content doesn't have any background color.

= I don't see the effect =
Try to define a unique ID for your footer and set the same ID in WE Footer Reveal, don't miss the # before the ID, and a point . before classes.

= The Plugin doesn't work as expected =
Please use the plugin forum, and open a new thread if your case was not already treated
